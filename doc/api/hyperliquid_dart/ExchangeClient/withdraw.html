Withdraw USDC to an external Ethereum address (via Arbitrum bridge).

Initiates an on-chain withdrawal from Hyperliquid to Arbitrum. Subject to network confirmations and bridge processing time.

Parameters:

  • destination - Ethereum address on Arbitrum (42-character hex)
  • amount - Amount in USDC as string (e.g., "100.5")

Example:

// Withdraw 500 USDC to external wallet
await exchange.withdraw(
  destination: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
  amount: '500.0',
);

Note: Withdrawals are processed on-chain and may take time to confirm. For internal transfers, use usdTransfer instead (instant and feeless).
