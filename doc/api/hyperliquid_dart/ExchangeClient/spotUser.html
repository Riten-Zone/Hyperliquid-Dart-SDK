Toggle spot dusting settings.

When optOut is true, prevents automatic dusting of small spot token amounts. Uses L1 action signing (msgpack + keccak256).

Example:

// Disable spot dusting
await exchange.spotUser(optOut: true);

  required bool optOut,
}) async {
  final action = <String, dynamic>{
    'type': 'spotUser',
