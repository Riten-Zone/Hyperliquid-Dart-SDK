WebSocketClient

WebSocketClient class

High-level WebSocket client for subscribing to Hyperliquid real-time data.

Wraps WebSocketTransport with typed subscription methods and automatic reconnection with re-subscription.

final ws = WebSocketClient();
await ws.connect();

final handle = ws.subscribeL2Book('BTC', (book) {
  print('BTC bids: ${book.bids.length}');
});

// Later:
await handle.cancel();
await ws.disconnect();
